# FinQA RAG Assistant

Система ответов на финансовые вопросы по документам. Решение строит ответ только на основе найденных фрагментов, сочетая поиск, переранжирование и генерацию.

## Задача
Нужно по коллекции финансовых документов находить фрагменты, достаточные для ответа на вопрос, и формировать ответ, опирающийся на найденные источники.

## Архитектура решения
```text
Финансовые документы
        │
        ▼
Разбиение на фрагменты
        │
        ▼
Векторизация
        │
        ▼
Поиск top-K фрагментов
        │
        ▼
Переранжирование
        │
        ▼
Генерация ответа по найденным фрагментам
```

## Структура репозитория
- `src/data_loader.py` — загрузка и подготовка документов;
- `src/embedding_service.py` — получение векторных представлений;
- `src/retriever.py` — поиск релевантных фрагментов;
- `src/generator.py` — формирование ответа;
- `src/pipeline.py` — полный конвейер обработки;
- `baseline.py` — упрощённый сценарий для сравнения;
- `main.py` — точка входа.

## Метрики
- `Recall@K` для этапа поиска;
- доля вопросов, для которых релевантный фрагмент найден в `top-K`;
- изменение качества после переранжирования;
- задержка по этапам;
- соответствие ответа найденным источникам.

## Результаты
В итоговом README должны быть указаны реальные значения:
- качество поиска до и после переранжирования;
- доля ответов, опирающихся на найденные фрагменты;
- задержка основных этапов;
- сравнение основного конвейера с `baseline.py`.

Формат записи:
- `Переранжирование повысило долю релевантных фрагментов в top-K с ... до ...`
- `Основной конвейер дал более устойчивые ответы по сравнению с базовым вариантом`
- `Средняя задержка полного прохода составила ...`

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python baseline.py
python main.py
```

## Ограничения и развитие
- Качество ответа ограничено качеством поиска.
- При длинных документах важны стратегия разбиения и размер `top-K`.
- Следующий шаг — добавить автоматический отчёт по качеству поиска и итоговых ответов.
